<h2 align="center">Project Setup 🚀</h2>

### 📄 Requisitos previos

- pnpm

### 📁 Clonar Repositorio

To use this project locally, run the following commands in your terminal:

```bash
git clone https://github.com/EricV29/exospace.git
cd exospace
pnpm install
```
